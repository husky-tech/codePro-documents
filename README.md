## 使い方

各コースで用いる画像はここから**ダウンロード**してご利用ください！

それぞれのコース毎にフォルダ分けしているので、フォルダ内に画像データが配置されています。
<br><br><br>

## 画像のダウンロード方法

右上の緑色のボタン「↓Code」をクリックし、  
「Download Zip」から Zip を PC にダウンロードし、
解凍してご利用下さい。
